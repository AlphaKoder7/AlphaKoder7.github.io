# Abin Issac Portfolio

Personal portfolio website, live at **[alphakoder7.github.io](https://alphakoder7.github.io)**

## About

Single page portfolio showcasing my work in cloud infrastructure, DevOps, and platform engineering. Highlights include the Phoenix Protocol automated disaster recovery pipeline on Azure and a containerised FastAPI job queue service.

## Stack

* Plain HTML, CSS, and vanilla JavaScript with no frameworks and no build step
* CSS variables for single point theming
* Fully responsive layout with mobile navigation
* Hosted on GitHub Pages

## Structure
index.html      page structure and content
style.css       all styling, theme variables in :root
script.js       smooth scroll, navbar blur, mobile menu
assets/         CV and images
